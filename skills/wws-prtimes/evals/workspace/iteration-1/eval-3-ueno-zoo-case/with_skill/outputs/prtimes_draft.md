# 上野動物園が MapPenguin を園内マップに採用。11言語でインバウンド来園者の動物案内を実現。

---

**リード文**

株式会社 World Wide System（東京都・以下 WWS）は、デジタルマップSaaS「MapPenguin」が東京都恩賜上野動物園（以下、上野動物園）の園内マップとして採用されたことを発表します。インバウンド来園者の増加に対応するため、動物の位置・説明・餌やり時間を11言語でリアルタイムに案内する仕組みを構築。アプリのダウンロード不要でQRコードをかざすだけで使えるシンプルな導線が採用の決め手となりました。

---

## ▼ 背景：訪日外国人の増加が施設案内に新たな課題

2024年以降、訪日外国人数は過去最高水準で推移しており、動物園・水族館・植物園など公共施設においても英語・中国語・韓国語をはじめとした多言語案内の整備が急務となっています。一方、従来の紙マップや館内サイネージでは言語バリエーションの維持・更新に多大なコストが発生し、リアルタイムな情報（餌やり時間の変更・臨時休園エリアなど）への対応も困難でした。

上野動物園は年間約350万人の来園者を誇る日本有数の動物園であり、外国人来園者比率の高まりに対して「低コストかつ即時更新可能な多言語デジタルマップ」を求めていました。

---

## ▼ MapPenguin とは

MapPenguin は、施設・観光地・工場・キャンパスなど「広い空間を案内する」あらゆるシーンに対応したデジタルマップSaaSです。地図上にスポット情報（写真・説明・動画・音声ガイド・営業時間）を登録するだけで、11言語対応のインタラクティブマップをWebブラウザ上に即時公開できます。

**サービスサイト**: https://maps.smartpr.jp/pro/

---

## ▼ 主な特長

| 特長 | 詳細 |
|------|------|
| アプリ不要 | URLを開くだけ。QRコードを読み取ればスマートフォンのブラウザで即時利用可能 |
| 11言語対応 | 日本語・英語・中国語（簡体・繁体）・韓国語など11言語に自動対応（Google翻訳連携） |
| リアルタイム更新 | 餌やり時間の変更・休園エリアの情報を管理画面から即時反映 |
| 音声ガイド対応 | 動物ごとに音声解説を登録。視覚障害のある方や子ども向け案内にも活用可能 |
| データ分析 | Looker Studio 連携でスポット別閲覧数・言語別利用率を可視化 |
| セキュリティ | ISO27001・プライバシーマーク取得済み |

## ▼ 導入事例：東京都恩賜上野動物園

### 導入企業プロフィール

| 項目 | 内容 |
|------|------|
| 施設名 | 東京都恩賜上野動物園 |
| 所在地 | 東京都台東区上野公園 9-83 |
| 年間来園者数 | 約350万人（2023年度実績） |
| 特徴 | 日本最古の動物園。約3,000頭の動物を飼育 |

### 導入前の課題

- 紙の園内マップは多言語版の印刷・配布に年間数百万円のコストが発生
- 餌やり時間・イベント情報の変更をリアルタイムで来園者に伝える手段がなかった
- 外国人来園者からの「動物の場所がわからない」という問い合わせが飼育員・スタッフの業務を圧迫

### 選定理由

MapPenguin が採用された最大の決め手は**インバウンド対応**の手軽さでした。アプリのインストールを来園者に求めず、入口ゲートのQRコードをかざすだけで多言語マップが起動する仕組みは、スマートフォンの操作に不慣れな外国人来園者にも受け入れやすいと評価されました。また、スポット情報の更新が管理画面から即時行えるため、飼育スタッフが餌やり時間を変更した際にも紙マップの刷り直しなく情報を最新化できる点が高く評価されました。

### 導入内容・活用方法

- 園内全エリアの動物展示スポット（約300か所）を MapPenguin 上に登録
- 各スポットに動物の説明テキスト・写真・餌やり時間スケジュールを11言語で掲載
- 入口・主要分岐点にQRコードを設置し、来園者が自身のスマートフォンでアクセス
- 担当者コメント（鎌田）による運用サポートを継続提供

### 担当者コメント

> 「上野動物園様のような年間数百万人規模の施設でも、管理画面の操作は非常にシンプルです。今回の導入により、スタッフの問い合わせ対応工数の削減とインバウンド来園者の体験向上を同時に実現できることを確信しています。」
>
> — 株式会社 World Wide System　鎌田
