## ▼ 背景：工場が抱えるマップ課題と安全管理の空白

日本の製造業では、構内案内や安全誘導に紙の地図や古い看板を使い続ける現場が多く残っています。外国人労働者の増加や工場レイアウトの変更のたびに印刷・配布コストが発生し、緊急時の誘導遅れや入構者の迷子問題が深刻化しています。

また、技術継承・新人教育の観点からも「構内のどこに何があるか」を視覚的・多言語で共有できるデジタルインフラの必要性が高まっています。しかし既存の地図システムは初期投資が高く、導入ハードルが企業規模を問わず大きな壁となっていました。

## ▼ MapPenguin Pro とは

**MapPenguin Pro** は、株式会社 World Wide Systemが提供するクラウド型デジタルマップSaaS（Software as a Service）です。工場・製造業の構内マップをスマートフォン・PCから閲覧できるインタラクティブなWebマップとして構築でき、**アプリのダウンロード不要・QRコード設置不要** で即日運用を開始できます。

| 特徴 | 詳細 |
|------|------|
| 多言語対応 | 11言語（Google翻訳連携）で外国人労働者にも対応 |
| セキュリティ | ISO27001・プライバシーマーク取得済み |
| 拡張性 | API / IoT / SSO 連携に対応 |
| サポート | 専任担当者による伴走支援 |
| 新機能（8月〜） | 通行止め管理・詳細権限管理を標準搭載予定 |

サービスサイト：https://maps.smartpr.jp/pro/

## ▼ 主な特長

### 1. 導入の簡便さ——アプリ不要・QRコード不要
URLを共有するだけで社内外から地図にアクセスできます。スマートフォンのブラウザで動作するため、作業員への端末配布や専用アプリのインストールは不要です。

### 2. 11言語対応で外国人労働者をカバー
日本語・英語・中国語・韓国語・ベトナム語など11言語に対応。外国人技能実習生・特定技能労働者が増加する製造業現場でも、言語の壁なく構内案内を提供します。

### 3. 安全管理・緊急誘導への対応
スポット情報の登録（危険箇所・避難経路・消火器設置場所など）や音声ガイドに対応。2026年8月以降は通行止め管理・リアルタイムルート更新も標準搭載予定です。

### 4. データ分析による継続改善
Looker Studio 連携により、地図上のアクセスヒートマップや閲覧数データを可視化。どのエリアへの案内ニーズが高いかを定量的に把握できます。

### 5. 高いセキュリティ基準
ISO27001認証・プライバシーマーク取得済み。工場内の機密情報を含む地図データを安全に管理します。限定公開URL発行機能で、外部公開範囲のコントロールも可能です。
