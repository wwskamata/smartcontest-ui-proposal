# 上野動物園がMapPenguinを採用。多言語で動物案内を実現。（32字 ✅）

---

## リード文

株式会社World Wide Systemは、上野動物園（東京都台東区）へのデジタルマップSaaS「MapPenguin」の採用を発表します。園内マップとして活用し、動物の位置・説明・餌やり時間を多言語で来園者に案内します。インバウンド需要の高まりを受け、多言語対応の即時実現が選定の決め手となりました。

（152字）

---

## ▼ 背景：インバウンド急増と多言語案内の課題

近年、訪日外国人旅行者数の回復とともに、動物園・水族館・植物園などの公共施設でもインバウンド対応への需要が急増しています。しかし従来の紙製マップや案内板は言語対応の更新コストが高く、展示内容の変更や餌やりタイムなどのリアルタイム情報を多言語で反映することが困難でした。

上野動物園は年間来園者数が300万人を超える国内最大級の動物園であり、外国人来園者の割合も年々増加しています。言語の壁を取り除き、すべての来園者に等しく情報を届けるためのデジタル施策が求められていました。

---

## ▼ MapPenguin とは

MapPenguin（マップペンギン）は、株式会社World Wide Systemが開発・運営するデジタルマップSaaSです。施設の地図上に動物・展示物・設備の位置情報・説明文・スケジュールなどを紐付け、スマートフォンブラウザから閲覧できるインタラクティブな案内システムを、専門知識なしで構築・運用できます。

- **サービスURL**：https://maps.smartpr.jp/pro/
- **提供形態**：SaaS（月額サブスクリプション）
- **主な対象**：動物園・水族館・テーマパーク・博物館・工場・商業施設など

---

## ▼ 主な特長

| 特長 | 詳細 |
|------|------|
| 多言語対応 | 管理画面から登録した情報を複数言語で即時表示。言語追加の都度、システム改修は不要 |
| リアルタイム更新 | 餌やり時間・休止情報・混雑状況などを管理画面から即座に反映 |
| アプリ不要 | QRコードを読み込むだけでブラウザ上でマップを表示。来園者はアプリをダウンロードする必要なし |
| カスタムピン設計 | 動物・施設・イベントごとにアイコンをカスタマイズし、直感的な案内を実現 |
| 管理者ダッシュボード | 閲覧数・人気スポット・言語別アクセスなどの統計データをリアルタイムで確認可能 |

## ▼ 上野動物園 導入事例

### 導入企業プロフィール

| 項目 | 内容 |
|------|------|
| 施設名 | 上野動物園（東京都恩賜上野動物園） |
| 所在地 | 東京都台東区上野公園9-83 |
| 年間来園者数 | 約300万人（都内最大級の動物園） |
| 特徴 | ジャイアントパンダをはじめとする約3,000頭の動物を展示。国内外から幅広い年齢層が訪れる |

### 導入前の課題

- 紙製マップは印刷コストが高く、展示変更のたびに改訂が必要
- 多言語版の制作・配布コストが課題となり、外国人来園者への案内が不十分
- 餌やり時間・イベント情報の更新が間に合わず、現地スタッフへの問い合わせが集中
- インバウンド来園者から「自分の言語で案内がほしい」という声が増加

### 選定理由

MapPenguinはアプリのインストールを必要とせず、QRコードをスキャンするだけで多言語の案内を即座に表示できます。管理者が専門知識なしでコンテンツを更新できるため、現場スタッフの運用負担も最小化されます。**インバウンド対応を即時実現できる点が最終的な決め手**となりました。

### 導入内容・活用方法

- **園内マップのデジタル化**：全展示エリア・施設の位置をインタラクティブマップ上に掲載
- **動物情報の多言語表示**：動物名・特徴説明を多言語で提供。来園者が自国の言語で閲覧可能
- **餌やり時間のリアルタイム案内**：当日のスケジュールを管理画面から随時更新
- **QRコードによるアクセス**：園内各所のサイネージ・パンフレットにQRコードを設置
