## SmartStampCard（WWS）

### 概要
1つのスポットで何度でもスタンプを取得できるデジタルスタンプカード。
QRコード・GPS・シリアルを組み合わせてスタンプ取得条件を設定できる。
業種・規模を問わず利用可能。

### スタンプ取得方式（4パターン・標準機能）
| 方式 | 内容 |
|------|------|
| QRコード | スポットのQRを読み取る |
| GPS | 位置情報でスポット範囲内を判定 |
| QR＋GPS | 両方を同時に満たす |
| シリアルコード | コードを手入力 |

### 主な標準機能
- GPS位置情報取得・スポット登録
- スタンプグリッド表示（カレンダーアイコン型）
- マイページ（ポイント・スタンプ・応募履歴）※マイレージOP時
- 賞品応募・当選管理
- 参加前アンケート＋応募時フォーム
- スポット1日あたり取得制限
- メール配信

### オプション機能と費用
| オプション | 追加初期費用 |
|-----------|------------|
| LINEログイン | +5万円 |
| マイレージ機能 | +5万円（ライト） / +10万円（スタンダード・カスタマイズ） |
| クーポン配布 | +10万円 |
| デジタルギフト配布 | +5万円 |
| インスタントウィン | +10万円 |
| LINE UID提供 | +5万円（LINEログイン利用時のみ） |

### プラン料金（税抜）
| プラン | 初期費用 | 月額費用 | 納期 | 最低契約 |
|--------|---------|---------|------|---------|
| スタンダード | 250,000円 | 30,000円 | 10日 | 1カ月 |
| カスタマイズ | 要問合せ | 要問合せ | 2カ月〜 | 3カ月 |

### 実UIの特徴（PDF確認済み）
- 背景：薄いミントグリーン（`#eaf6f2`）
- スタンプ：カレンダーアイコン型の丸スタンプ（数字入り）
  - 取得済：`#4caf50`（緑）
  - 未取得：`#e0e0e0`（グレー）
- ボタン：赤（`#c8373a`）
- アクセントカラー：オレンジ（`#e87722`）
- ヘッダー：ロゴ（W）＋タイトルのみのシンプル構成
- LINEログインボタン：`#06C755`（LINE公式グリーン）

## SmartRecieco（WWS）

### 概要
レシート応募／シリアル応募でマストバイキャンペーンを強力にサポート。
LINE版・WEB版の両方に対応。

### 応募方式
| 方式 | 内容 |
|------|------|
| レシート応募 | レシート画像をアップロード→OCR自動審査 |
| シリアル応募 | シリアルコードを入力 |

### OCRエンジン
- **Gemini OCR / LINE OCR** を採用
- 読取項目：店舗名・日時・金額・商品名・支払方法など
- 精度は100%ではないため一部人手審査が必要

### 担当者（事務局）フロー
```
投稿収集 → OCR自動審査（OK/待ち/NG分類）→ 担当者検閲（NGのみ）→ 当選者決定 → 当選通知
```

### 重複チェック
- **同一応募者内のリアルタイム重複チェック**：あり（標準）
- **全応募者横断の重複チェック**：バッチ処理（日次設定可）
  - 判定条件：日時（分単位）＋店舗名＋金額が一致
  - バッチ実行中は処理速度が低下する場合あり

⚠️ **OCR結果はユーザー画面には表示されない**（管理画面のみ）

### プラン料金（WEB版・税抜）

| カスタマイズ | 60万円 | 6万円 | 2カ月〜 |

**LINE版はWEB版より初期+20〜30万円高め**

## SmartStampRally（WWS）

### 概要
GPSスタンプラリーシステム。各スポットで1回のスタンプ取得。
SmartStampCardとの違いは「同一スポットで何度でも」ではなく「各スポット1回限り」。

### 主な特徴
- GPS・QRコードでのスポット判定
- インターバル制限（同一スポットの再取得制限）
- ルート設定・コンプリートボーナス
- ラリーコース設定

---

## プロダクト選定チャート

```
ポイントを何度でも貯めたい（同一場所に繰り返し来訪）
  → SmartStampCard（マイレージ）

特定スポットを1回ずつ巡るラリー形式
  → SmartStampRally

レシート・領収書での購買証明が必要
  → SmartRecieco

GPS判定＋レシート判定を1キャンペーンで統合したい
  → SmartStampCard（GPS）＋ SmartRecieco（Receico）カスタム連携
  → ※標準機能では連携不可。カスタム開発が必要
```
