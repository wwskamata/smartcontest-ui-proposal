## SmartStampCard（WWS）

### 概要
1つのスポットで何度でもスタンプを取得できるデジタルスタンプカード。
QRコード・GPS・シリアルを組み合わせてスタンプ取得条件を設定できる。

### スタンプ取得方式（標準機能4パターン）
| 方式 | 内容 |
|------|------|
| QRコード | スポットのQRを読み取る |
| GPS | 位置情報でスポット範囲内を判定 |
| QR＋GPS | 両方を同時に満たす |
| シリアルコード | コードを手入力 |

### 主な標準機能
- GPS位置情報取得・スポット登録
- スタンプグリッド表示（カレンダーアイコン型）
- マイページ（ポイント・スタンプ・応募履歴）※マイレージOP時
- 賞品応募・当選管理
- 参加前アンケート＋応募時フォーム
- スポット1日あたり取得制限・インターバル制限

### オプション機能と費用
| オプション | 追加初期費用 |
|-----------|------------|
| LINEログイン | +5万円 |
| マイレージ（ライト） | +5万円 |
| マイレージ（スタンダード） | +10万円 |
| クーポン配布 | +10万円 |
| デジタルギフト配布 | +5万円 |
| インスタントウィン | +10万円 |
| LINE UID提供 | +5万円（LINEログイン利用時のみ） |

### プラン料金（税抜）
| プラン | 初期費用 | 月額費用 | 納期 | 最低契約 |
|--------|---------|---------|------|---------|
| スタンダード | 250,000円 | 30,000円 | 10日 | 1カ月 |
| カスタマイズ | 要問合せ | 要問合せ | 2カ月〜 | 3カ月 |

### 実UIの特徴
- 背景：薄いミントグリーン（`#eaf6f2`）
- スタンプ：カレンダーアイコン型の丸スタンプ（数字入り）
  - 取得済：緑（`#4caf50`） / 未取得：グレー（`#e0e0e0`）
- ボタン：赤（`#c8373a`）
- アクセントカラー：オレンジ（`#e87722`）
- LINEログインボタン：LINE公式グリーン（`#06C755`）

## SmartRecieco（WWS）

### 概要
レシート応募でマストバイキャンペーンを強力にサポート。LINE版・WEB版両対応。

### 応募方式
| 方式 | 内容 |
|------|------|
| レシート応募 | レシート画像アップロード → OCR自動審査 |
| シリアル応募 | シリアルコードを入力 |

### OCRエンジン
- Gemini OCR / LINE OCR 採用
- 読取項目：店舗名・日時・金額・商品名・支払方法など
- ⚠️ **OCR結果はユーザー画面に表示しない（管理側のみ）**

### 担当者（事務局）フロー
```
投稿収集 → OCR自動審査（OK/待ち/NG分類）→ 担当者検閲（NGのみ）→ 当選者決定 → 当選通知
```

### 重複チェック
- 同一応募者内のリアルタイム重複チェック：あり（標準）
- 全応募者横断の重複チェック：日次バッチ（日時＋店舗名＋金額一致で判定）

### プラン料金（WEB版・税抜）

| カスタマイズ | 60万円 | 6万円 | 2カ月〜 |

**LINE版はWEB版より初期+20〜30万円高め**

## SmartStampRally（WWS）

### 概要
GPSスタンプラリーシステム。各スポットで1回のスタンプ取得（SmartStampCardとの違い）。

### 特徴
- GPS・QRコードでのスポット判定
- インターバル制限（同一スポットの再取得制限）
- ルート設定・コンプリートボーナス
- ラリーコース設定

## MapPenguin（WWS）

### 概要
デジタルマップSaaS（施設・工場・観光地向け）。QRコード不要・URLシェアのみで利用可。

| 項目 | 内容 |
|------|------|
| URL | https://maps.smartpr.jp/pro/ |
| 主要用途 | 工場構内マップ・商業施設マップ・観光マップ・キャンパスマップ |

### プラン（ベースプラン・月額税別）
| プラン | 月額 | 初期費用 | スポット上限 |
|--------|------|---------|------------|
| Standard | ¥8,000 | ¥100,000 | 100件 |
| Pro | ¥23,000 | ¥100,000 | 300件 |
| Pro plus | ¥38,000 | ¥100,000 | 500件 |
| Enterprise | ¥72,000〜 | ¥100,000 | 1,000件〜 |

全プラン共通：利用ユーザー無制限・11言語対応・Looker Studio・専任担当者

### オプション機能
| オプション | 費用 | 体系 |
|-----------|------|------|
| 経路案内 | ¥20,000 | 月額 |
| GTFSデータ連携 | ¥24,000 | 月額 |
| モデルコース | ¥15,000 | 月額 |
| 機械翻訳 | ¥15,000 | 初期 |
| ヒートマップ | ¥10,000 | 月額 |

### 工場向けの主な訴求ポイント
1. アプリ不要・QRコード不要（URLシェアのみ）
2. 11言語対応で外国人労働者もカバー
3. ISO27001・プライバシーマーク取得（セキュリティ）
4. API/IoT/SSO連携対応
5. 専任担当者が伴走サポート
6. 通行止め・緊急誘導対応（8月以降 標準搭載予定）

### CSV一括登録
- スポット情報をCSVで一括インポート可能
- GTFSデータなしでも運用可（停留所座標を直接CSV登録）

## Admin-DX CMS（Kuroco）

### 製品概要
| 項目 | 内容 |
|------|------|
| エンジン | Kuroco（ヘッドレスCMS）|
| カテゴリ | SaaS型ヘッドレスCMS・APIファーストCMS |
| 問い合わせ | https://ww-system.com/wws/service/wws_kuroco/inquiry/ |

### 製品の特徴（4本柱）
| 特徴 | 説明 |
|------|------|
| ヘッドレス構成 | フロントエンド（Next.js等）とCMSを分離。デザイン・技術選定が自由 |
| APIファースト | REST / GraphQL API + Webhook 標準装備。外部システム連携が容易 |
| マルチテナント | 複数サイト・複数組織を1基盤で管理 |
| スケーラビリティ | CDNキャッシュ活用により高負荷に対応 |

### 競合との差別化
| 比較軸 | Admin-DX（Kuroco） | WordPress | 従来型CMS |
|--------|------------------|-----------|---------|
| フロント自由度 | ◎ ヘッドレス | △ テーマ依存 | ✕ |
| API連携 | ◎ 標準装備 | △ プラグイン必要 | ✕ |
| スケーラビリティ | ◎ CDN対応 | △ サーバー依存 | △ |
| セキュリティ | ◎ SaaS自動管理 | △ 自己管理 | △ |
| マルチサイト管理 | ◎ 標準対応 | △ | ✕ |

### 主なユースケース
| 業種 | 用途例 |
|------|--------|
| メディア・放送 | 番組サイト・記事CMS（岡山放送OHK実績） |
| 不動産・物件管理 | 物件検索プラットフォーム・賃貸情報サイト |
| 起業支援・会員制 | 会員管理・予約・情報配信サイト（アントレサポート案件） |
| 企業サイト | グループサイト統合・多言語コーポレートサイト |

### 料金（要問合せ）
| コスト項目 | 備考 |
|-----------|------|
| 初期構築費 | フロントエンド開発費 + Kuroco設定費 |
| Kuroco月額 | SaaS月額。規模・機能によりプラン変動 |
| 保守・運用費 | 必要に応じてオプション |

## 重要な横断ルール

| ルール | 内容 |
|--------|------|
